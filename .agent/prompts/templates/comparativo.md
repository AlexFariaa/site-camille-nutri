# Template: Comparativo

**Quando usar:** Artigos que comparam ferramentas, plataformas, abordagens ou estratégias. Alta intenção de decisão — o leitor está em dúvida e quer ajuda para escolher. Funciona bem para termos de busca do tipo "X ou Y", "melhor ferramenta para Z".
**Exemplos de título:** "X vs Y: qual é melhor para [profissão]?", "Comparativo: [ferramenta A] ou [ferramenta B] para [contexto]?"

---

## REGRA CRÍTICA DE TÍTULOS

**Os H2s devem nomear os ITENS COMPARADOS e os CRITÉRIOS reais — não fórmulas genéricas.** O risco aqui é o modelo usar sempre "o critério que decide tudo" ou "quando escolher X ou Y" sem adaptar ao tema.

Exemplos do que NÃO fazer:
- ❌ "Por que essa escolha importa" — genérico
- ❌ "O critério que ninguém menciona" — poderia ser qualquer comparativo
- ❌ "Quando escolher X e quando escolher Y" — vago

Exemplos do que FAZER:
- ✅ "WhatsApp Business vs CRM: por que a maioria das clínicas usa o errado para agendamento"
- ✅ "Doctoralia x Google Meu Negócio: quem realmente traz mais pacientes para psicólogos?"
- ✅ "O critério que as clínicas ignoram ao escolher sistema de prontuário: integração com WhatsApp"
- ✅ "Se você tem até R$300/mês: qual plataforma de agendamento vale mais para dentistas?"

## Estrutura

```
Gancho (2-3 parágrafos)
  → A dúvida real e específica que o leitor tem na cabeça sobre ESTES dois itens
  → Por que escolher errado tem consequências práticas para aquele nicho
  → Promessa: ao fim do artigo, a escolha vai ser clara — com critérios objetivos

H2: [Título que nomeia POR QUE ESTA ESCOLHA É CRÍTICA para este profissional]
  → O custo concreto de usar a opção errada neste contexto específico
  → Critérios que o leitor geralmente ignora (específicos ao tema, não genéricos)
  → Dado ou situação que ilustra o problema de escolher mal

H2: [Título que apresenta OS ITENS COMPARADOS — com nome real de cada um]
  → Descrição honesta de cada opção: o que é, para quem foi feita, o que promete
  → Pontos fortes e fracos de cada um — sem viés
  → Contexto de uso: quando cada um brilha

H2: [Título que nomeia O COMPARATIVO DIRETO com os critérios específicos]
  → Use <table> com colunas: Critério / [Item A] / [Item B]
  → Critérios relevantes para o nicho deste artigo (não critérios genéricos)
  → Seja honesto: se um item perde em algo, diga claramente
  → Adicione coluna de "quem precisa disso" por critério

H2: [Título que nomeia O PERFIL DE ESCOLHA — quem deve escolher qual]
  → Cenários práticos com nome: "se você é [perfil A], escolha X porque [razão específica]"
  → Evite resposta genérica — qualifique por orçamento, tamanho, especialidade, objetivo
  → Pode incluir uma terceira opção para quem está em um perfil intermediário

H2: [Título que nomeia O FATOR DECISIVO — o critério que mais pesa neste contexto]
  → O insight que só quem já usou os dois sabe
  → Específico ao contexto do artigo — não "o que ninguém menciona" genérico
  → Referência natural à A² Company se for pertinente

H2: [CTA — encoraja decisão com suporte]
  → Não empurra uma opção, mas abre caminho para quem quer ajuda para decidir
```

---

## Orientações de imparcialidade

- Apresente os dois lados com honestidade — leitores percebem quando o comparativo é propaganda disfarçada
- A recomendação final pode favorecer a solução da A², mas o comparativo em si deve ser justo
- Se um item é claramente superior para determinado perfil, diga — não relativize para parecer neutro
- Evite adjetivos vagos como "robusto", "poderoso", "completo" — use critérios mensuráveis

## Fórmulas de H2 — adapte aos itens comparados reais

```
[H2 de por que importa]
Fórmula: "Por que a maioria dos [profissão] escolhe [item errado] — e o que isso custa"
Fórmula: "[Item A] ou [Item B]: a escolha que define como seu consultório vai crescer"
Fórmula: "O erro de [profissão] ao comparar [Item A] e [Item B] sem olhar para [critério específico]"

[H2 de apresentação]
Fórmula: "[Item A] e [Item B]: o que cada um entrega de verdade (sem marketing)"
Fórmula: "O que é [Item A] e o que é [Item B] — antes de comparar, entenda o que cada um resolve"
Fórmula: "[Item A] vs [Item B]: propósitos diferentes, confusão na hora de escolher"

[H2 do comparativo]
Fórmula: "[Item A] x [Item B]: custo, usabilidade e resultado para [profissão]"
Fórmula: "Comparativo direto: [critério 1], [critério 2] e [critério 3] entre [A] e [B]"
Fórmula: "A tabela que [profissão] deveria ter antes de contratar [A] ou [B]"

[H2 do perfil]
Fórmula: "[Item A] para quem [perfil A] — [Item B] para quem [perfil B]"
Fórmula: "Quando [Item A] é a escolha certa (e quando não é)"
Fórmula: "O perfil de clínica que se arrepende de escolher [Item A] sem ver [Item B]"

[H2 do fator decisivo]
Fórmula: "O critério que define entre [A] e [B] para [profissão]: [critério específico real]"
Fórmula: "Depois de [N] comparativos, o que realmente separa [A] de [B] para [nicho]"

[CTA]
Fórmula: "Ainda em dúvida entre [A] e [B]? Veja o que faz sentido para o seu caso"
Fórmula: "[A] ou [B]: a decisão é sua — mas não precisa ser no escuro"
```
