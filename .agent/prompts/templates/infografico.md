# Template: Infográfico

**Quando usar:** Artigos estruturados para serem visualmente escaneáveis, com dados, estatísticas ou processos apresentados de forma condensada e visual. Excelente para backlinks, compartilhamento e SEO de featured snippets. O texto serve como contexto e análise dos dados — não como único conteúdo.
**Exemplos de título:** "[N] dados sobre [tema] que todo [profissão] precisa conhecer", "[Tema] em números: o panorama completo para [nicho]"

---

## REGRA CRÍTICA DE TÍTULOS

**Cada H2 deve nomear o GRUPO DE DADOS ou DIMENSÃO específica analisada — não rótulos genéricos de "dados" ou "números".** O erro mais comum é usar H2s como "Os dados revelam" ou "O que os números dizem" sem especificar o que foi revelado.

Exemplos do que NÃO fazer:
- ❌ "[Tema] em números: o que os dados revelam sobre [nicho]" — genérico demais como H2
- ❌ "O que esses números dizem sobre o futuro do seu consultório" — poderia ser qualquer artigo
- ❌ "[Categoria]: dados que mudam a perspectiva" — sem identidade do dado

Exemplos do que FAZER:
- ✅ "Presença digital de médicos no Brasil: o abismo entre quem tem site e quem depende só de indicação"
- ✅ "Agendamento online em clínicas: 73% dos pacientes preferem marcar sem ligar — mas poucos consultórios oferecem"
- ✅ "O que psicólogos perdem sem Google Meu Negócio: dados de visibilidade local que assustam"
- ✅ "Automação no atendimento: o que clínicas que reduziram no-show têm em comum"

## Estrutura

```
Gancho (2-3 parágrafos)
  → Abre com O DADO MAIS IMPACTANTE do artigo — na primeira linha
  → Contextualiza por que esses dados importam especificamente para este profissional
  → Tom: analítico mas acessível — não é relatório acadêmico, é panorama prático

H2: [Título que nomeia O CENÁRIO GERAL com dado central deste tema]
  → Bloco de 4-6 estatísticas relevantes com <ul> ou sequência destacada
  → Cada dado: o número + o que ele significa na prática (não só o número solto)
  → Blockquote com o dado mais impactante do bloco
  → Fontes qualificadas quando possível (IBGE, CFM, CFP, CRN, Google, pesquisas)

H2: [Título que nomeia DIMENSÃO 1 — primeiro ângulo dos dados, específico ao tema]
  → Grupo de dados sobre uma dimensão específica (ex: comportamento do paciente online)
  → Use <table> se houver comparação entre grupos, períodos ou categorias
  → Cada dado com interpretação: "o que isso significa para quem atende [nicho]"
  → 1-2 parágrafos de análise para cada bloco de dados
  → Placeholder de imagem inline aqui

H2: [Título que nomeia DIMENSÃO 2 — segundo ângulo, diferente do anterior]
  → Grupo de dados sobre outra dimensão (ex: adoção de tecnologia, conversão, presença digital)
  → Mesmo padrão: dado + interpretação + implicação prática
  → Varie o formato: se o anterior usou <ul>, use <table> aqui ou vice-versa

H2: [Título que nomeia A IMPLICAÇÃO PRÁTICA — o que esses dados exigem do profissional]
  → Síntese analítica: o que esses dados coletivamente revelam sobre o tema
  → 3-5 ações concretas que os dados sugerem (derivadas dos números, não genéricas)
  → Referência natural à A² Company como solução para o que os dados indicam

H2: [CTA — conecta dados à ação imediata]
  → Encerramento que transforma o panorama em urgência
```

---

## Orientações sobre dados

- **Prefira dados reais e verificáveis**: IBGE, CFM, CFP, CRN, Google Trends, pesquisas publicadas
- **Se não houver dado exato**: use "pesquisas indicam", "estudos apontam" — nunca invente estatísticas com precisão falsa
- **Cada dado precisa de contexto**: "73% dos pacientes" sem interpretar o que isso significa para o leitor é dado desperdiçado
- **Evite dados desatualizados**: não use estudos de 5+ anos sem mencionar que são referências históricas
- **Hierarquia de impacto**: coloque o dado mais surpreendente logo no gancho e repita-o no índice/H2 relevante

## Fórmulas de H2 — adapte aos dados e dimensões reais

```

Fórmula: "[Métrica específica] em [nicho]: o que separa quem cresce de quem estagna"
Fórmula: "O impacto de [ação/ferramenta] em [resultado]: o que os dados de [fonte] mostram"

[H2 de implicação prática]
Fórmula: "O que esses dados exigem de [profissão] que quer [resultado específico]"
Fórmula: "[N] mudanças que os dados sugerem para [nicho] em [tema]"
Fórmula: "Da leitura à ação: o que fazer com esses números no seu consultório"

[CTA]
Fórmula: "Os dados mostram o caminho — o que falta para [profissão] seguir?"
Fórmula: "[Dado principal do artigo]: isso já era o suficiente para agir. O que está esperando?"
```
