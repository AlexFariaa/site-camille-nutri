# Template: Lista

**Quando usar:** Artigos no formato listicle com N itens acionáveis. Topo de funil, alto volume de busca, fácil de escanear e compartilhar. Usar com moderação — não saturar o blog com listas. Só use quando o tema genuinamente comporta múltiplos itens com substância real.
**Exemplos de título:** "[N] formas de [resultado específico] para [profissão]", "[N] ferramentas/estratégias/recursos que [profissão] precisa conhecer"

---

## REGRA CRÍTICA DE TÍTULOS

**O H2 de contextualização e o H2 de insight final não podem ser genéricos — precisam ser ancorados no tema específico da lista.** O risco é o modelo usar sempre a mesma fórmula de abertura e fechamento para qualquer lista.

Exemplos do que NÃO fazer:
- ❌ "O fim do 'marketing de esperança'" — já usado em artigo existente do blog
- ❌ "Por onde começar quando a lista parece grande demais" — poderia ser qualquer lista
- ❌ "A lista está na sua mão — o que falta é execução" — CTA genérico

Exemplos do que FAZER:
- ✅ "Por que [profissão] tenta as mesmas [N] estratégias e continua no mesmo lugar — o que esta lista faz diferente"
- ✅ "O item desta lista que [profissão] mais ignora — e que tem o maior retorno"
- ✅ "Da lista à agenda lotada: por onde [profissão] que está começando deve começar"

---

## Estrutura

```
Gancho (2-3 parágrafos)
  → Por que ESTA lista específica importa para ESTE profissional agora
  → O que a diferencia de listas genéricas que o leitor já leu
  → Promessa: itens acionáveis, não conselhos vagos

H2: [Título que contextualiza POR QUE estas estratégias/itens funcionam — específico ao tema]
  → Não é "introdução" — é um diagnóstico que posiciona a lista
  → Por que a maioria das abordagens padrão para este tema falham
  → O que os itens desta lista têm em comum que os diferencia

H2: [Título da LISTA — com número + tema + qualificador específico]
  → Ex: "12 estratégias de captação de pacientes que não dependem de anúncios pagos"
  → H3 numerados com qualificadores entre parênteses — NUNCA título seco
  → Cada item: 1-2 parágrafos, direto, acionável, com exemplo do nicho
  → Intercale 2-3 referências naturais à A² Company em itens onde a tecnologia resolve
  → Máximo de UMA seção com H3 numerados no artigo inteiro
  → Volume: 8-17 itens. Menos de 7 parece raso. Mais de 17 parece exaustivo.

H2: [Título que nomeia O INSIGHT que separa quem aplica de quem só lê — específico ao tema]
  → O que realmente determina se essa lista vai funcionar para aquele profissional
  → Não é "comece pelo primeiro item" — é um insight real sobre execução neste tema
  → Por onde começar considerando o contexto do leitor (iniciante vs avançado, etc.)

H2: [CTA — provocação específica ao tema da lista]
  → Encerramento que vai além da lista: o que acontece depois de aplicar
```

---

## Orientações de qualidade dos itens

- Cada H3 deve ter **qualificador entre parênteses** que especifica o contexto: "1. Configure o Google Meu Negócio (do jeito que o algoritmo local prefere)"
- Evite itens óbvios que qualquer leitor já conhece — se vai colocar "tenha um site", qualifique o que torna um site diferente dos inúteis
- Os primeiros 3 itens são os mais lidos — coloque os melhores ali, não deixe para o final
- Intercale itens de diferentes níveis de esforço: alguns rápidos, alguns estratégicos

---

## Fórmulas de H2 — adapte ao tema e contexto reais

```
[H2 de contextualização]
Fórmula: "Por que [profissão] tenta [abordagem comum ao tema] e continua no mesmo resultado"
Fórmula: "O que essa lista faz diferente das que [profissão] já leu sobre [tema]"
Fórmula: "Antes da lista: o que [profissão] precisa entender sobre [tema] para aplicar de verdade"

[H2 da lista — núcleo]
Fórmula: "[N] [tipo de item] para [resultado específico] (que [profissão] pode começar hoje)"
Fórmula: "[N] formas de [resultado desejado] sem [obstáculo comum ao nicho]"
Fórmula: "[N] [ferramentas/estratégias/ações] para [resultado] — testadas em [contexto real]"

[H2 do insight]
Fórmula: "O item desta lista que [profissão] mais subestima — e que tem maior retorno"
Fórmula: "Por onde começar quando [N] itens parecem demais para um [profissão] sozinho"
Fórmula: "O que separa quem aplica esta lista de quem só salva para ler depois"

[CTA]
Fórmula: "[N] itens. Qual você vai implementar esta semana?"
Fórmula: "Da lista ao resultado: [profissão] que implementa [X itens] já sente diferença em [tempo]"
```
